# python-network-sniffer
A real-time network packet analyzer built using Python and Scapy to capture, inspect, and analyze network traffic including IP, TCP, UDP, and ICMP protocols.


Real-Time Network Packet Analyzer

A Python-based **Network Sniffer** built using **Scapy** to capture and analyze live network traffic. This project demonstrates how data flows through a network and how different protocols operate in real time.

Features

📡 Captures real-time network packets
🌐 Displays **source and destination IP addresses**
🔎 Identifies protocols (**TCP, UDP, ICMP**)
🔗 Maps port numbers to services (HTTP, HTTPS, DNS, etc.)
📄 Extracts and displays **payload data**
📊 Tracks packet statistics (TCP, UDP, ICMP counts)
🧠 Helps understand **network communication and protocols**

Technologies Used

Python  
Scapy  
Networking (TCP/IP)
